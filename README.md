# KeenerSentinel Releases

Official public release channel for KeenerSentinel.

KeenerSentinel is a personal Windows guard that watches autoruns, scheduled tasks, browser policies, and suspicious PowerShell/script execution. It is not a full antivirus; it is a helper tool for blocking account-stealer persistence patterns.

## Download

Use the latest release:

https://github.com/henerd/KeenerSentinel-Releases/releases/latest

Download `KeenerSentinel-Portable.zip`, extract it, then run `Install.cmd`.

## Update

Installed packages include `Update.cmd` and a daily scheduled update check. Updates are downloaded from this release channel.

## Verify

Each release includes `checksums.txt`. The SHA256 value in that file should match `KeenerSentinel-Portable.zip`.

Only use files from this official GitHub Release page.
